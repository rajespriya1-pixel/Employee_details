# Employee_details
# Overview

The Employee_Details Excel file contains essential employee information used for HR and workforce analysis. It acts as a foundational dataset for performing data cleaning, reporting, and creating HR dashboards.

# Dataset

The file includes key employee attributes such as:

Employee ID & Name

Department

Salary Details

Date of Joining (DOJ)

Experience Information

Performance Ratings

These fields help in understanding workforce structure, compensation distribution, employee performance, and departmental trends.
